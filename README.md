# Website CelleboyStore – Aplikasi Topup Game Online
**Aditya Hafizh Nugraha – Teknik Informatika – Universitas Pamulang**

Website CelleboyStore adalah aplikasi berbasis web yang digunakan untuk berbelanja atau yang biasa dibilang topup game online.

Aplikasi ini dikembangkan menggunakan PHP Native dan MySQL dengan tampilan antarmuka sederhana, terstruktur, dan mudah digunakan sebagai proyek pembelajaran pemrograman web.

---

## Tujuan Pengembangan

- Mempermudah proses top up game secara online
- Menyediakan sistem transaksi yang cepat dan mudah digunakan
- Mengelola data user dan riwayat transaksi
- Menerapkan konsep CRUD, autentikasi, dan database
- Sebagai media pembelajaran pemrograman web

---

### Fitur - Fitur Pengguna

- Registrasi akun pengguna
- Login dan logout sistem
- Memilih game
- Melakukan top up
- Melihat riwayat transaksi

---

## Alur Sistem

1. Pengguna melakukan registrasi akun
2. Pengguna login ke dalam sistem
3. Sistem menampilkan dashboard utama
4. Pengguna memilih game
5. Pengguna melakukan top up
6. Sistem menyimpan data transaksi ke database
7. Pengguna dapat melihat riwayat transaksi
8. Pengguna logout dari sistem
